# Simulador de Post

Projeto simples para simular a criacao de um post no estilo Facebook/LinkedIn. O usuario informa um titulo e um texto, envia o formulario e o resultado aparece no preview após um POST para a API do JSONPlaceholder.

## Funcionalidades
- Formulario com titulo e conteudo.
- Envio via fetch (POST) para https://jsonplaceholder.typicode.com/posts.
- Renderizacao do titulo e do texto retornados pela API.
- Layout responsivo com preview do post.

## Tecnologias usadas
- HTML5 (estrutura e semantica)
- CSS3 (layout, tipografia e responsividade)
- JavaScript (DOM, eventos e fetch)

## Como usar
1. Abra o arquivo index.html no navegador.
2. Preencha o titulo e o conteudo.
3. Clique em "Enviar agora" para ver o resultado no preview.

## Estrutura de arquivos
- index.html
- styles.css
- script.js
- image/favicon.svg
